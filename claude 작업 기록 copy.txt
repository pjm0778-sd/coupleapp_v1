## 현재 상태
- Flutter Web app, Supabase 백엔드
- GitHub Actions + Cloudflare Pages 배포 중 (npx wrangler@3 방식으로 변경됨)

## DB 스키마 핵심 정보
- color_mappings 컬럼: id, user_id, color_hex, work_type (title/end_time/start_time 없음!)
- schedules 컬럼: id, user_id, couple_id, date, title, work_type, start_time, end_time, category, color_hex, location, note, reminder_minutes, repeat_pattern, is_anniversary
- schedule_comments 컬럼: id, schedule_id, user_id, content, created_at (RLS 적용됨)
- couples 컬럼: id, user1_id, user2_id, started_at
- profiles 컬럼: id, nickname, couple_id
